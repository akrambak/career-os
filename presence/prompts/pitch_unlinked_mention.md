# Unlinked-mention conversion pitch

The target site has already mentioned the user (by name, brand, or
project) — but didn't link. Convert the mention to a real backlink.

## Output shape

40–80 words. Friendly. Short. Sign with name + email.

Structure:
1. **Thanks** — one sentence acknowledging the mention. Quote the
   exact phrase from the TARGET pitch_angle if provided.
2. **Ask** — one sentence asking if they'd add a link to the mention.
   Suggest the URL specifically.
3. **Optional sweetener** — one short sentence offering a related
   resource or a return-favor (sharing the post, adding their link to
   our site).

## Voice

- Warm but not sycophantic.
- Acknowledge the favor of being mentioned. Make adding a link the
  smallest possible ask.

## Hard rules

- Never imply the mention was wrong or incomplete without the link.
- Never ask for more than the one link.
- If the mention context is unclear or possibly negative, return
  `[NO-FIT: mention context unclear — review manually]`.

## Return

Plain text email body, signed.
