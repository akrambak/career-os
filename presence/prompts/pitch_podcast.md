# Podcast pitch

Cold outreach from **Bakhouche Akram (AkBak)** to a podcast host asking
to guest. 8y production fullstack, now layering Claude SDK on top.

## Output shape

70–120 words. No subject. Sign with name + email.

Structure:
1. **Hook** — name an episode you'd reference. If TARGET pitch_angle
   provides one, use it. If not, write topic-themed (e.g. "I noticed
   you cover AI tooling for solo devs").
2. **Why me, why now** — one sentence: production reality of building
   AI agents on top of an established stack, not just LLM demos.
3. **Topic** — one sentence proposing what you'd talk about. Specific
   ("how I wired Claude SDK into a Laravel monolith without breaking
   the AB-test framework") beats vague ("AI in production").
4. **Ask** — "Happy to record any week — DM me a calendar link." Then
   name + email.

## Voice

- First-name basis. Podcasts are warm.
- Specific topic. Hosts get spammed with "I can talk about AI" pitches.
- One sentence of bio max. Don't dump credentials.

## Hard rules

- Never invent past employers, customers, metrics.
- If the podcast is a poor fit, return `[NO-FIT: <reason>]`.

## Return

Plain text. No subject.
