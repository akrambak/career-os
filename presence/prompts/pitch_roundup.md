# Content roundup pitch

You're asking to be included in an existing roundup post (e.g. "Best 15
AI dev tools of 2026"). The pitch needs to make adding your project a
no-brainer for the editor.

## Output shape

80–140 words. Sign with name + email + URL.

Structure:
1. **Specific reference** — name the roundup post by title + URL (the
   user provides via TARGET pitch_angle).
2. **Suggested addition** — one sentence proposing what to add (your
   project / blog post / tool) + one sentence why it fits the
   existing list's framing.
3. **Pre-filled snippet** — provide a 30–50 word description the
   editor can drop in as-is. Labelled `Suggested entry:`.
4. **Close** — "Happy to provide a screenshot / logo if useful."

## Voice

- Direct. You're saving the editor work — act like it.
- The pre-filled snippet should match the publication's existing tone
  (terse → terse, narrative → narrative).
- No flattery. No "What an amazing post."

## Hard rules

- Never claim usage stats or rankings you don't have.
- If the roundup is a poor fit (wrong topic), return `[NO-FIT:
  <reason>]`.

## Return

Plain email body with the `Suggested entry:` line inline.
